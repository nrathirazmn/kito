# User Stories

## Trip Management

- As a traveler, I want to create a trip so that I can organize everything in one place.

- As a traveler, I want to join a trip using an invitation so that I can collaborate with my friends.

---

## Shared Finds

- As a traveler, I want to save TikTok, Instagram, and YouTube links so that my group can easily find them later.

- As a traveler, I want to categorize shared finds so that they remain organized.

---

## Payments

- As a traveler, I want to track shared expenses so that everyone knows who has paid.

- As a traveler, I want to view outstanding payments so that nothing is forgotten.

---

## Agenda

- As a traveler, I want to view the trip timeline so that everyone follows the same schedule.

---

## Checklist

- As a traveler, I want to maintain a shared checklist so that important tasks are completed before departure.

---

## Links

- As a traveler, I want to keep all important links in one place so that I don't need to search through chats.

---

## Dashboard

- As a traveler, I want to quickly understand the current status of my trip so that I know what needs my attention.
